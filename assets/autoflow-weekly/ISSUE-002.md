## The ROI Framework 📊

Before we start, here's how we calculated:

**Hourly Value = Annual Revenue ÷ 2,000 hours**

(Assuming 50 weeks × 40 hours, minus vacation/holidays)

**Tool ROI = (Time Saved × Hourly Value) - Monthly Cost**

Example: If you bill $100/hour and a tool saves 5 hours/month:
- Value created: $500
- Tool cost: $20
- **Net ROI: $480/month**

Simple math. Powerful results.
